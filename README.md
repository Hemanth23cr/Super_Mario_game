# Super_Mario_game

## Super Mario Game coded in Python

I used to play Super Mario game on video_game more on my childhood. To remember those memories I have coded this game on python.

## Dependencies

   * $ Pip install pygame
   * $ pip install scipy
    
## Controls

 * Left: Move left
 * Right: Move right
 * Space: Jump
 * Shift: Boost
 * Left/Right Mouseclick: secret
